This project includes a Douyin web API client, Cookie handling flow, X-Bogus signing code, msToken handling, and related request patterns adapted from or inspired by public Apache-2.0 licensed implementations used in the douyin-downloader / Douyin-TikTok downloader ecosystem.

The Cookie login flow follows the same user-controlled pattern commonly used by open-source Douyin tools: Playwright opens the Douyin website, the user logs in manually, and cookies are saved locally.
