# Notice

This project includes a Douyin web API client and X-Bogus signing code adapted from public Apache-2.0 licensed implementations used in the Douyin/TikTok downloader ecosystem.

The Cookie login flow follows the same user-controlled pattern commonly used by open-source Douyin tools: Playwright opens the Douyin website, the user logs in manually, and cookies are saved locally.

Users are responsible for complying with Douyin's terms, local laws, and the rights of content owners.
